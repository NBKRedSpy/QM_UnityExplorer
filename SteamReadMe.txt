[h1]Quasimorph Unity Explorer[/h1]


Unity Explorer for Quasimorph

Uses https://github.com/NBKRedSpy/UnityExplorer-Standalone-Harmony which is a fork of https://github.com/sinai-dev/UnityExplorer

[h1]Configuration[/h1]

[h2]Unity Explorer Specific Configuration[/h2]

The Unity Explorer's configs will be created at <steam dir>\steamapps\workshop\content\2059170\3494516652

[h2]MCM[/h2]

This mod optionally supports the Mod Configuration Menu.

[h2]Config File[/h2]

The configuration file will be created on the first game run and can be found at [i]%AppData%\..\LocalLow\Magnum Scriptum Ltd\Quasimorph_ModConfigs\MapMarkers\config.json[/i].
[table]
[tr]
[td]Name
[/td]
[td]Default
[/td]
[td]Description
[/td]
[/tr]
[tr]
[td]Enable
[/td]
[td]True
[/td]
[td]Enables the UE.  If false, UE will not be initialized and has no impact on the game whatsoever.
[/td]
[/tr]
[/table]
